### 导入方法
- 1.下载[Nebula on the Ship](https://github.com/Dolly1016/Nebula)模组并且正确导入，正确运行一次。
- 2.在游戏目录下应新建的**Addons**目录加入本项目的压缩包文件。  
*注:不要解压本压缩包文件*
- 3.如果没有出现差错，那么导入完毕。
