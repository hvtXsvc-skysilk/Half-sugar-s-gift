# Contributing to Half Sugar's Gift

首先，感谢你花时间贡献代码！这份指南会告诉你如何参与到本项目中来。

## 我该如何提供帮助？

### 1. 报告 Bug
如果你发现了 Bug，请提交至开发者，并尽可能包含以下信息：
- **描述**：发生了什么，你预期会发生什么。
- **复现步骤**：详细地列出复现的步骤。
- **截图 / 日志**：如果可以，请附上相关的截图或日志（游戏根目录下的GiftLog和NebulaLog两个文件），这非常有帮助。

### 2. 提出新想法
我们欢迎新的想法。
- **想实现的事情**：你是想以怎样的类型实现?实现一个什么?职业或者指令亦或者其他?
- **希望的解决方案**：你希望这个功能如何工作？

### 3. 贡献代码
在开始编写代码前，**建议先创建一个 Issue 讨论**，特别是对于比较大的改动。

#### 代码提交流程 (PR)
1. **Fork 本仓库**：将本仓库 Fork 到你自己的 GitHub 账号下。
2. **Clone 你的仓库**：`git clone https://github.com/你的用户名/项目名.git`
3. **创建新分支**：
   ```bash
   git checkout -b feat/your-feature-name

- 如果你不想这么做，那也可以向开发者直接提交代码文件。

#### 如何联系开发者?
- 你可以查看[项目介绍文档](README.md)来得知我们的QQ群。加入他来报告你发现的bug或者想要提交的创意。
- 或者，你可以联系**hvtXsvc@outlook.com**。我们会在采纳后为您发送回信。
